# ReceitApp

> App Flutter de receitas culinarias com Firebase e timer de cozinha.

## Sobre o projeto

Aplicativo mobile de receitas desenvolvido em Flutter com Firebase Authentication e Cloud Firestore. Cada usuario gerencia receitas e listas personalizadas com CRUD completo, busca, validacao de formularios e timer de cozinha com alarme sonoro. Interface Material 3 com tema claro/escuro, imagens por URL e navegacao por abas.

## Funcionalidades principais

- Login, cadastro e logout com Firebase Auth
- CRUD de receitas com ingredientes e etapas numeradas
- CRUD de listas com receitas vinculadas
- Timer de cozinha com alarme sonoro e vibracao
- Tema claro/escuro e UI Material 3 responsiva
- Busca, ordenacao e navegacao com GoRouter

## Tecnologias utilizadas

- Flutter
- Dart
- Firebase Auth
- Cloud Firestore
- Provider
- GoRouter
- Material 3

## Como executar

cd ReceitasApp; flutter pub get; flutterfire configure; flutter run

## Deploy / Demonstracao

App mobile - build APK via flutter build apk.
